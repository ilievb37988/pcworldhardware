The GeForce RTX 3080 Ti is an enthusiast-class graphics card by NVIDIA, launched on May 31st, 2021. This ensures that all modern games will run on GeForce RTX 3080 Ti. Additionally, the DirectX 12 Ultimate capability guarantees support for hardware-raytracing, variable-rate shading and more, in upcoming video games. The card also has 80 raytracing acceleration cores. NVIDIA has paired 12 GB GDDR6X memory with the GeForce RTX 3080 Ti, which are connected using a 384-bit memory interface. The GPU is operating at a frequency of 1365 MHz, which can be boosted up to 1665 MHz, memory is running at 1188 MHz (19 Gbps effective).
